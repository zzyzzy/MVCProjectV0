JSP 페이지 모듈화
일반적으로 웹 페이지는 동일한 상/하단 영역과
메뉴에 따라 달라지는 메인영역으로 구성되어 있음

이런 경우 JSP액션 태그를 이용해서 웹페이지를
모듈화시키면 코드가 중복되는 것을 방지할 수 있음

즉, 동일한 상/하단 영역은 jsp:include액션태그로
구성하고 메뉴에 따라 메인영역에 보여질 부분만
jsp 페이지로 지정하는 방식을 사용함 => 템플릿 사용

포워드 vs 리다이렉트

리다이렉트redirect : 클라이언트에게 지정된
                    url로 재요청하도록 지시함
    ex) 고객이 상담원에게 전화를 걸었는데
        특정번호로 다시 거시라고 안내하는 경우

포워드forward : 클라이언트의 요청을 받은 JSP가
           직접 처리하지 않고 다른 JSP에게 처리하라고 위임함
    ex) 고객이 상담원에게 전화를 걸었는데
        상담원이 문의사항을 몰라서 팀장에게 물어보고 응답함
